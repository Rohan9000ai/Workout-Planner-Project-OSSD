 GymFit – Workout Plan Application
An open-source desktop/web application for personalized gym workout planning, fitness tracking, and exercise guidance.

📌 About the Project
GymFit is a fitness application developed as part of an Open Source Software Development course. It helps users track their health metrics, choose a fitness goal (Fat Loss or Strength Training), and follow structured exercise routines with guided workouts.

 
Features
User Signup & Login with profile management
Real-time health metrics display (calories, steps, heartbeat, height, weight)
Goal-based workout modules: Fat Loss and Strength Training
Exercise counter with guided reps/sets for fat loss workouts
Muscle-specific exercise library with images for strength training
Feedback system and easy logout from the menu bar
Scraped data integration for calorie tracker and step counter
